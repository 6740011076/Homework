สร้างการ์ดโปรไฟล์ด้วย CSS

โจทย์

Level : Beginner

โจทย์นี้จะให้คุณฝึกสร้างและจัดรูปแบบการ์ดโปรไฟล์ง่ายๆ โดยใช้ HTML และ CSS เพื่อแสดงข้อมูลส่วนตัว เช่น รูปภาพ ชื่อ และข้อมูลติดต่อ การ์ดโปรไฟล์นี้จะช่วยให้คุณได้ฝึกใช้คุณสมบัติพื้นฐานของ CSS เช่น border, box-shadow, text styling และการจัดวางองค์ประกอบ
สิ่งที่โจทย์ต้องการ:
• ทำให้การ์ดโปรไฟล์มีขอบ (border) และเงา (box-shadow) ที่สวยงาม
• จัดกึ่งกลางการ์ดโปรไฟล์บนหน้าเว็บทั้งแนวตั้งและแนวนอน
• ทำให้รูปโปรไฟล์เป็นวงกลมสมบูรณ์
• เปลี่ยนสีข้อความของชื่อและตำแหน่งงานให้แตกต่างกัน เพื่อให้อ่านง่ายและน่าสนใจ
• ปรับระยะห่างภายใน (padding) ของการ์ด และระยะห่างภายนอก (margin) ระหว่างองค์ประกอบต่างๆ ภายใน เพื่อให้อ่านง่ายและดูเป็นระเบียบ
• เปลี่ยนฟอนต์ (font-family) และขนาดตัวอักษร (font-size) ของชื่อ, ตำแหน่งงาน และข้อมูลติดต่อ ให้ดูเหมาะสมและแตกต่างกัน
