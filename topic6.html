การ์ดข้อมูลพื้นฐาน

โจทย์

สร้างการ์ดแสดงข้อมูลสินค้าหรือบริการ โดยใช้ HTML สำหรับโครงสร้างและ CSS เพื่อจัดรูปแบบให้สวยงาม น่าอ่าน และมีลักษณะเด่นออกมาจากพื้นหลัง
สิ่งที่โจทย์ต้องการ:
• กำหนดความกว้างของการ์ด (.card) เป็น 300px.
• กำหนดสีพื้นหลังของการ์ดเป็นสีขาว (#ffffff).
• เพิ่มระยะห่างภายใน (padding) ของการ์ด 20px.
• ทำให้มุมของการ์ดโค้งมนเล็กน้อยด้วย border-radius: 8px.
• เพิ่มเงาให้กับการ์ดด้วย box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1).
• กำหนดสีข้อความของหัวเรื่อง (h3) ภายใน .card เป็นสีน้ำเงินเข้ม (#336699).
• จัดกึ่งกลางข้อความทั้งหมดภายใน .card (text-align: center).
