สร้างการ์ดข้อมูลสินค้า

โจทย์

คุณได้รับมอบหมายให้สร้างการ์ดข้อมูลสินค้าแบบง่ายๆ โดยใช้ HTML และ CSS หน้าที่ของคุณคือการจัดรูปแบบการ์ดให้ดูสวยงามและใช้งานง่าย โดยเน้นการใช้คุณสมบัติ CSS พื้นฐานเพื่อควบคุมขนาด สี การจัดวาง และเงา
สิ่งที่โจทย์ต้องการ:
• สร้างการ์ดสินค้าที่มีความกว้างประมาณ 250-300px และมีการจัดกึ่งกลางในแนวนอนและแนวตั้งของหน้าจอ
• เพิ่มขอบโค้งมน (border-radius) และเงา (box-shadow) ให้กับการ์ดเพื่อให้น่าสนใจยิ่งขึ้น
• จัดรูปแบบรูปภาพสินค้าให้มีขนาดเหมาะสมและอยู่ด้านบนของการ์ด
• จัดรูปแบบชื่อสินค้า (h3) และราคา (p) ให้มีขนาดตัวอักษรและสีที่อ่านง่าย
• จัดรูปแบบปุ่ม 'เพิ่มลงตะกร้า' ให้มีพื้นหลังและสีตัวอักษรที่แตกต่าง และมี padding ที่เหมาะสม
