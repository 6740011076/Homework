สร้างการ์ดโปรไฟล์ด้วย CSS

โจทย์

ในโจทย์นี้ คุณจะต้องสร้างการ์ดโปรไฟล์ที่สวยงามโดยใช้ HTML และ CSS เพื่อแสดงรูปภาพ ชื่อ และตำแหน่งงาน การ์ดจะถูกจัดให้อยู่กึ่งกลางของหน้าเว็บและมีรูปแบบที่น่าสนใจ
สิ่งที่โจทย์ต้องการ:
• ทำให้การ์ดโปรไฟล์ (`.profile-card`) มีพื้นหลังเป็นสีขาว (`background-color`)
• เพิ่มขอบโค้งมน (`border-radius`) ให้กับการ์ดโปรไฟล์
• เพิ่มเงาให้กับการ์ดโปรไฟล์ (`box-shadow`) เพื่อให้ดูมีมิติ
• จัดข้อความทั้งหมดภายใน `.profile-card` ให้อยู่กึ่งกลาง (`text-align`)
• กำหนดขนาดรูปโปรไฟล์ (`.profile-image`) ให้เป็นสี่เหลี่ยมจัตุรัสขนาด 120x120px
• ทำให้รูปโปรไฟล์เป็นวงกลม (`border-radius: 50%`)
• เพิ่มระยะห่างภายใน (`padding`) ให้กับการ์ดโปรไฟล์
• เปลี่ยนสีข้อความของชื่อ (`.profile-name`) และตำแหน่ง (`.profile-title`) ให้เหมาะสม
