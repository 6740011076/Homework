สร้างตัวนับจำนวนพื้นฐาน

โจทย์

Level : Beginner

คุณต้องสร้างหน้าเว็บง่ายๆ ที่มีตัวนับจำนวน เมื่อผู้ใช้กดปุ่ม 'เพิ่ม' ตัวเลขจะเพิ่มขึ้น และเมื่อกดปุ่ม 'ลด' ตัวเลขจะลดลง ค่าเริ่มต้นของตัวนับคือ 0
สิ่งที่โจทย์ต้องการ:
• สร้างไฟล์ HTML ที่มีส่วนแสดงผลตัวเลข (ใช้ id="counter-display") และปุ่มสองปุ่ม (ใช้ id="decrement-btn" และ id="increment-btn")
• ใช้ JavaScript เพื่อจัดการค่าตัวนับ โดยเริ่มต้นที่ 0
• เมื่อกดปุ่มที่มี id="increment-btn" ตัวนับควรเพิ่มขึ้นทีละ 1
• เมื่อกดปุ่มที่มี id="decrement-btn" ตัวนับควรลดลงทีละ 1
• อัปเดตค่าที่แสดงผลใน element ที่มี id="counter-display" ให้ตรงกับค่าตัวนับปัจจุบัน
